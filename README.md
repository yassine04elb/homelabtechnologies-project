# Mijn Homelab Project

Dit homelab gebruik ik om mijn kennis in systeem- en netwerkbeheer te oefenen.

## Wat ik heb opgezet
- **pfSense firewall/router** – configureren en beheren
- **Windows Server (DC)** – Active Directory, DNS en Domain Controller configureren en beheren
- **Gebruikersbeheer** – users, groups, Organizational Units en Group Policies instellen en beheren
- **File Server** – opzetten en beheren

## Skills die ik oefen
- Installatie, configuratie en troubleshooting van systemen
- Netwerkbeheer en beveiliging
- Windows Server en AD beheer

## Volgende stappen
- Update Server opzetten
- Monitoring Server configureren en beheren
